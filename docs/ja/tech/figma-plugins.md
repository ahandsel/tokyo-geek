Figmaはライティングに最適なツールとは言えませんが、今やデザインの業界標準です。UXライターに役立つFigmaプラグインを紹介します。

| プラグイン                       | 説明                                                                  |
| -------------------------------- | --------------------------------------------------------------------- |
| [Find and Replace Layer Names][] | レイヤー名をワンクリックで検索・置換できるシンプルなツールです。      |
| [Select Layers][]                | 名前、タイプ、類似性に基づいてレイヤーを選択できます。                |
| [Phrase Strings][]               | Phrase StringsからUXコピーをレイヤー名に紐づけてFigmaに取り込めます。 |
| [Change Text][]                  | 複数のレイヤー、グループ、フレームのテキストを一括編集できます。      |
| [Rename It][]                    | レイヤーやフレームを一括リネームできます。                            |
| [Reset Text Layer Name][]        | テキストレイヤーの名前をコンテンツに合わせてリセットできます。        |


## 使用例とベストプラクティス


### Find and Replace Layer Names

[Find and Replace Layer Names][]プラグインを使うと、Figmaファイル内のレイヤー名を素早く検索・置換できます。複数のレイヤー名を一度に更新したいときに特に便利です。例えば、一貫した命名規則のデザインシステムがある場合、このプラグインでファイル全体の整合性を保てます。


### Select Layers

[Select Layers][]プラグインを使うと、名前、タイプ、類似性などの条件に基づいてレイヤーを選択できます。特定のタイプのレイヤーを一括変更したいときに役立ちます。例えば、すべてのテキストレイヤーを選択して新しいフォントスタイルを適用できます。


### Phrase Strings

[Phrase Strings][]プラグインを使うと、Phrase StringsとFigma間でUXコピーをプル・プッシュできます。デザインが常に最新のコピーと同期されるようにするのに便利です。コピーをレイヤー名に紐づけることで、手動でコピー＆ペーストせずにデザイン内のテキストを簡単に更新できます。

このプラグインは、関連するテキストレイヤーの名前が正しく設定されていて、少数のレイヤーだけを扱う場合にうまく動作します。
テキストレイヤーが多いとFigmaがフリーズしたりクラッシュしたりすることがあるので、注意してください。


### Change Text

[Change Text][]プラグインを使うと、複数のレイヤー、グループ、フレームのテキスト内容を一括編集できます。大量のテキストを変更する必要がある場合に時間を大幅に節約できます。例えば、複数の画面にわたるボタンの文言を更新したいとき、このプラグインで素早く効率的に作業できます。


### Rename It

[Rename It][]プラグインを使うと、レイヤーやフレームを一括リネームできます。Figmaファイルを整理し、一貫した命名規則を維持するのに便利です。例えば、レイヤー名にプレフィックスやサフィックスを追加したり、レイヤー名内の特定の単語を置換したりできます。


### Reset Text Layer Name

[Reset Text Layer Name][]プラグインを使うと、テキストレイヤーの名前をそのコンテンツに合わせてリセットできます。レイヤーを整理し、Figmaファイル内の特定のテキストレイヤーを見つけやすくするのに役立ちます。例えば、コンテンツが「送信」のテキストレイヤーがある場合、このプラグインでレイヤー名を「送信」にリネームできます。
