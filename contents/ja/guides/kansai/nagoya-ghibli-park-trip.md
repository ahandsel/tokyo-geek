## ジブリパーク周辺のコインロッカー


### 注意：ジブリパーク内にコインロッカーはありません

コインロッカーは愛・地球博記念公園内にありますが、ジブリパーク自体の中にはありません。


### コインロッカーの利用時間

すべてのコインロッカーは 9:00 から 18:00 まで利用できます。

> [!IMPORTANT] コインロッカーは 18:00 に閉まります。それまでに必ず荷物を取り出してください！


### 周辺のコインロッカー

1. **コインロッカー（北口案内所） / Coin Locker (North Entrance Information)**
   * **場所：** ジブリパークの青春の丘エリア付近。
   * **料金とサイズ：**
     * 小：¥300 (W35.2cm x H33.5cm x D67.8cm)
     * 中：¥500 (W35.2cm x H57.6cm x D67.8cm)
     * 大：¥600 (W35.2cm x H86.2cm x D67.8cm)
   * **数量：** 小：37、中：15、大：4

2. **コインロッカー（西口案内所） / Coin Locker (West Entrance Information)**
   * **場所：** ジブリパークのジブリの大倉庫エリアに比較的近い場所。
   * **料金とサイズ：**
     * 小：¥300 (W35.2cm x H33.5cm x D67.8cm)
     * 中：¥500 (W35.2cm x H57.6cm x D67.8cm)
     * 大：¥600 (W35.2cm x H86.2cm x D67.8cm)
   * **数量：** 小：21、中：6、大：2

3. **クローク（北案内所） / Cloak (North Information)**
   * **場所：** 青春の丘エリア付近。ロッカーに入らない大きな荷物用です。
   * **料金とサイズ：** ¥600、20kg まで、どの辺も 180cm まで
   * **備考：** 貴重品はお預かりできません。現金払いのみ。18:00 までに引き取りが必要です。

詳しくは[こちらの記事](https://castel.jp/p/9201)をご覧ください。
