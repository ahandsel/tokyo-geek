# ふるさと納税 - 故郷納税制度


## 用語集

* 給与収入（`給与収入`, きゅうよしゅうにゅう）
  * 給与収入とは、社会保険料や所得税などの控除前の、雇用による総額（額面）の収入を指します。
* 給与所得（`給与所得`, きゅうよしょとく）
  * 給与所得とは、給与収入から給与所得控除額を差し引いた金額です。
  * [給与所得] = [給与収入] - [給与所得控除額]
  * [給与所得] = [給与収入] - [給与所得控除額]
* 給与所得控除額（`給与所得控除額`, きゅうよしょとくこうじょがく）
* 控除限度額（`控除限度額`, こうじょげんどがく）
  * 控除限度額とは、ふるさと納税で控除される最大額のことです。
* 源泉徴収票（`源泉徴収票`, げんせんちょうしゅうひょう）
  * 源泉徴収票は、雇用主が発行する、従業員の年間の収入と源泉徴収された税額を記載した書類です。
* 額面（`額面`, がくめん）
  * 額面とは、税金や社会保険料などの控除前の総収入を指します。
