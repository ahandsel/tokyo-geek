## Travel time between Tokyo and Nagoya

| Name   | JP       | Time     | Note                                       |
| ------ | -------- | -------- | ------------------------------------------ |
| Nozomi | `のぞみ` | 1.65 hrs | Fastest, Cannot use JR Pass                |
| Hikari | `ひかり` | 1.73 hrs | Can use JR Pass                            |
| Kodama | `こだま` | 2.6 hrs  | Slowest (1.05 hrs slower), Can use JR Pass |
